1. 搭建环境

1.1 开发环境

安装以下组件并配置环境变量

  • CentOS release 6.5 (Final) 64bit
  • git 2.10.0
  • Gradle 2.4
  • JDK 1.8
  • Android SDK Platform 23
  • Android Build Tools 23.0.02
  • Android Extras（Android support Library）

TIPS：

  • centos 64 bit 上buil-tools 中的aapt工具不能很好运行，需要安装
    • yum install libgcc.i686
    • yum install zlib.i686
    • yum install glibc.i686
    • …
