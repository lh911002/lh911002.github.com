Quick Response Code

发表于 2016-12-08 | 分类于 Common ， Quick-Response-Code |
  • 日本DENSO WAVE公司发明，应用最广泛的二维码之一，可以快速获取信息的内容，采用深色表示1，浅色表示0
  • 参考文档：http://www.qrcode.com/en/about/standards.html & ISO标准

1. 特点

  1. 扫描时不用完全对准二维码即可完成识别
  2. 支持多种内容类型，字符串，图片，视频等
  3. 纠错能力强
